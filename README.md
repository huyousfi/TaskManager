# Advanced Todo App

A full-stack Todo application built with Node.js, Express, MongoDB, and vanilla JavaScript.

## Features

- User Registration
- User Login
- JWT Authentication
- Password Hashing (bcrypt)
- Create Todo
- Edit Todo
- Delete Todo
- Complete Todo
- User-specific Todos
- Logout

## Tech Stack

### Frontend

- HTML
- CSS
- JavaScript

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt

## Installation

### Backend

```bash
npm install
npm start
```

### Frontend

Open `login.html` using Live Server.

## Environment Variables

Create a `.env` file:

```
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
